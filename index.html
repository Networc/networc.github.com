Welcome

This is the personal website and blog of Andrew C. Reed.

All current content deals with Keyshanc (KEYboard SHA-based eNCrypter), which is my attempt to thwart keyloggers on workstations where the user cannot install software. Keyshanc is open source - please +1 and Tweet (#keyshanc) about it if you feel that it advances web security.

Over time, I will continue to add my future open source projects and blog about them. Thanks for visiting!
